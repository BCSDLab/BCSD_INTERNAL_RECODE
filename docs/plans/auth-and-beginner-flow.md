# 인증 플로우 개선 + 비기너 지원 플로우

## 현재 상태

### 로그인
- Google OAuth로 로그인
- 가입된 계정 → 로그인 완료
- 미가입 계정 → 401 `"member not found, registration required"` → 회원가입 페이지로 리다이렉트

### 회원가입
- 4단계 위저드: Google 인증 → 정보 입력(이름, 학과, 학번, 전화번호) → 학교 이메일 인증 → 트랙 선택
- 로그인에서 리다이렉트 시 Google 인증 단계 스킵
- 하단에 "이미 계정이 있으신가요? 로그인" 링크

## 인증 플로우 개선

### 로그인 → 회원가입 리다이렉트 UX 개선
- 현재: 미가입 시 "로그인에 실패했습니다" 에러가 잠깐 보임
- 개선: 에러 대신 "가입이 필요합니다" 안내 후 자연스럽게 회원가입으로 이동
- 로그인 페이지 하단에 "처음이신가요? 회원가입" 링크 추가

### 회원가입 경고 배너
- 문제: 아무 Google 계정으로 가입하는 경우가 있음
- 해결: 회원가입 위저드 상단에 경고 배너 표시
  - "BCSD에서 계속 사용할 Google 계정으로 가입해주세요."
  - "다른 계정으로 가입하면 나중에 변경이 어렵습니다."
- 기존 회원은 회원가입 페이지를 거치지 않으므로 경고를 안 봄

### 이미 가입된 계정으로 회원가입 시도
- "이미 가입된 계정입니다" 안내 → 로그인 페이지로 유도

### 학교 이메일 기반 다중 Google 계정 연결 (장기)
- 회원의 고유 키를 학교 이메일로 변경
- 하나의 회원에 여러 Google 계정 연결 가능 (다대일)
- 다른 Google 계정으로 로그인해도 학교 이메일로 기존 회원과 매칭
- 이렇게 되면 경고 배너의 중요도가 낮아짐 (잘못된 계정으로 가입해도 나중에 연결 가능)

## 비기너 지원 플로우

### 전체 흐름

```
bcsdlab.com "지원하러 가기"
  │  (모집기간 아니면 "현재 모집 기간이 아닙니다" 표시)
  ▼
internal 로그인
  ├─ 기존 회원 → 로그인 완료 → 지원 페이지
  └─ 미가입 → "가입이 필요합니다" → 회원가입 (⚠️ 경고 배너)
       → 가입 완료 → 지원 페이지
  ▼
비기너 지원 페이지
  │  자체 폼 (구글폼 스타일 UI, DB 직접 저장)
  │  조건 분기:
  │    - 이미 활동 중인 회원 → "이미 활동 중인 회원입니다"
  │    - 이미 지원서 제출 → 지원 상태 페이지로 이동
  │    - 모집 기간 아님 → "현재 모집 기간이 아닙니다"
  │    - 지원 가능 → 폼 표시
  ▼
지원 완료 → 지원 상태 페이지
  │  상태: 검토 중 / 납부 대기 / 승인 / 반려
  ▼
[관리자/부회장] 지원자 목록 확인
  │  납부 확인 → 승인/반려
  ▼
승인 시:
  │  비기너로 상태 변경
  │  슬랙 온보딩 안내
  ▼
비기너 활동 시작
```

### 지원 폼
- 이름, 이메일, 학교 이메일 — 자동 채움, 수정 불가 (가입 시 입력한 값)
- 희망 트랙 — 자동 채움, 수정 가능 (가입 시 선택한 트랙)
- 지원 동기 — 필수
- 기타 질문 — 추후 관리자가 질문을 설정할 수 있으면 이상적

### 지원 상태
- 검토 중: "지원서를 검토 중입니다."
- 납부 대기: "회비 납부가 확인되면 승인됩니다." + 납부 안내 정보
- 승인: "BCSD 비기너로 승인되었습니다!" + 슬랙 온보딩 안내
- 반려: "이번에는 함께하지 못하게 되었습니다." + 사유(선택)

### 관리자 지원자 관리
- 지원자 목록: 이름, 이메일, 희망 트랙, 지원일, 상태
- 상태 변경: 검토 중 → 납부 대기 → 승인 / 반려
- 일괄 승인/반려 가능

### 슬랙 온보딩
- Slack 무료 플랜이라 자동 초대 API 사용 불가
- 승인 시 슬랙 초대 링크를 지원 상태 페이지에 표시
- 추후 유료 플랜 전환 시 자동 초대로 업그레이드 가능

### 모집 기간 관리
- 관리자가 모집 시작/종료일 설정
- bcsdlab.com과 지원 페이지에서 모집 기간 여부에 따라 분기
- 모집 기간 아닐 때 직접 URL 접근해도 "모집 기간이 아닙니다" 표시
